## 一、登入後台

**步驟：**

1. 用手機或電腦打開瀏覽器
2. 輸入網址：`https://bni-changrong66-yiseadata.zeabur.app/admin`
3. 畫面出現「管理員後台」登入頁
4. 輸入管理員密碼（由技術窗口提供）
5. 點選「**進入管理後台**」按鈕

> ⚠️ 若出現「密碼錯誤，請重試」→ 確認密碼是否輸入正確，密碼有區分大小寫

## 二、認識主畫面：5 個數字的意義

登入後，畫面最上方會看到 **5 個紅色數字**，代表目前系統的即時狀況：

```
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│    25    │    18    │    42    │    30    │    12    │
│  參與者  │  使用者  │  總媒合  │  想認識  │  可提供  │
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

| 數字名稱 | 意義 | 說明 |
|---------|------|------|
| **參與者** | 系統內的與會名單總人數 | 上傳 CSV 後才會出現數字，活動前應與名單人數一致 |
| **使用者** | 實際登入系統的人數 | 活動進行中會持續增加，代表有幾個人已打開網站 |
| **總媒合** | 所有人標記的次數總和 | 我想認識他 + 我可以幫助他 的加總 |
| **想認識** | 大家按下「我想認識他」的次數 | 越多代表互動越熱絡 |
| **可提供** | 大家按下「我可以幫助他」的次數 | 代表媒合配對的潛在機會 |

> 💡 點選數字下方「**重新整理數據**」可手動更新這 5 個數字

## 三、活動前：上傳參與者名單

> 建議在活動開始前 30 分鐘完成這個步驟

**所需檔案格式：**
- 檔案類型：`.csv`（Excel 可另存為此格式）
- 欄位順序：`name,industry,table,needs`（第一列為標題，不可更改）
- 編碼：UTF-8（建議使用 Google 試算表匯出）

**欄位說明：**

| 欄位 | 中文意義 | 範例 | 必填 |
|------|---------|------|------|
| `name` | 姓名 | 王小明 | ✅ 必填 |
| `industry` | 行業 | 室內設計 | ✅ 必填 |
| `table` | 桌號 | 3 | ✅ 必填 |
| `needs` | 商務上需要的協助 | 需要法律顧問 | 可空白 |

**操作步驟：**

1. 在主畫面找到「📤 匯入參與者名單」區塊
2. 點選「📁 點擊選擇 CSV 檔案」
3. 從手機或電腦選擇準備好的 CSV 檔案
4. 確認下方出現「已選擇：檔名.csv」
5. 點選「**上傳並匯入**」（紅色按鈕）
6. 等待出現「✅ 成功匯入 XX 筆資料」
7. 確認上方「參與者」數字與名單人數吻合

> ⚠️ 若匯入失敗，請確認 CSV 格式是否正確，或聯絡技術窗口

## 四、活動中：切換目前發言者

> 每次換人上台報告時操作，讓所有會員即時看到目前誰在說話

**操作步驟：**

1. 在主畫面向下捲動，找到「👥 所有參與者」區塊
2. 在搜尋欄位輸入發言者的**姓名或桌號**快速找到該人
3. 找到後，點選該人右側的「**設為發言者**」按鈕
4. 按鈕變為紅色「發言中」→ 操作成功
5. 同時，畫面上方「🎤 目前發言者」區塊會顯示該人的名字

**確認方式：**
- 畫面上方紅色區塊出現：`🎤 LIVE｜王小明｜室內設計｜第3桌`
- 所有會員手機上會在 5 秒內自動更新，看到相同資訊

> 💡 每次只能有一位發言者，設定新的發言者會自動取消上一位

---

## 五、活動中：清除發言者

> 中場休息、活動結束、或暫時不需要顯示發言者時使用

**操作步驟：**

1. 找到「🎤 目前發言者」區塊
2. 點選「**🔕 清除目前發言者**」按鈕（灰色框線按鈕）
3. 上方顯示「目前尚無發言者」→ 操作成功
4. 所有會員手機畫面的發言者區塊會在 5 秒內消失

---

## 六、活動後：下載媒合報表

> 活動結束後操作，下載所有媒合紀錄

**操作步驟：**

1. 在主畫面找到「📊 匯出媒合報表」區塊
2. 點選「**📥 下載 CSV 報表**」（綠色按鈕）
3. 檔案自動下載，檔名格式：`bni-connections-2026-04-22.csv`
4. 用 Excel 或 Google 試算表開啟即可閱讀

**報表內容欄位說明：**

| 欄位 | 意義 |
|------|------|
| 使用者姓名 | 標記操作的會員姓名 |
| 使用者身份 | 長榮會員 / 來賓 / 親友 |
| 標記對象 | 被標記的參與者姓名 |
| 標記類型 | 想認識 / 可提供 |
| 時間 | 標記發生的時間點 |

## 八、注意事項與禁止操作

### ✅ 正常操作
- 上傳名單、設定發言者、清除發言者、下載報表

### ⛔ 活動中禁止操作（除非確認無誤）

| 按鈕 | 後果 |
|------|------|
| 👥 清除所有使用者資料（含媒合記錄） | 所有人的登入紀錄與媒合標記全部消失，**無法復原**。參與者名單保留。 |
| 🗑️ 清除所有參與者資料（含媒合記錄） | 名單與所有媒合紀錄全部消失，**無法復原**。使用者帳號保留。 |

> ⚠️ 以上兩個按鈕僅在需要「重新開始」時使用，正常操作不需要用到
>
> 💡 **清除使用者**適合在同一份名單要重複辦兩場活動時使用；**清除參與者**則用於更換全新名單
